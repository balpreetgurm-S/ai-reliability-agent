# AI Reliability Agent

A production-style agentic AI system for incident investigation that helps engineering teams analyze alerts using logs, metrics, deployment history, and runbook intelligence.

## Highlights
- Tool-based AI investigation workflow
- Root cause analysis with cited evidence
- Traceable agent reasoning and observability
- Evaluation metrics for response quality
- Modern React + FastAPI architecture

## Tech Stack
- Frontend: React + TypeScript + Tailwind
- Backend: FastAPI
- Database: SQLite
- AI Layer: Tool-calling agent workflow (mock provider)
